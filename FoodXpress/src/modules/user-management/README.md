# User Management Module - Tanushka

This module handles user profile management and user-related operations.

## Structure
- `components/` - User management UI components
- `hooks/` - Custom hooks for user operations
- `services/` - User API services
- `utils/` - User-related helper functions
- `styles/` - Module-specific styles

## Usage
```jsx
import { UserProfile, useUser } from '@/modules/user-management';
```

## Components
- UserProfile
- UserList
- UserSettings

## Hooks
- useUser
- useUserProfile
