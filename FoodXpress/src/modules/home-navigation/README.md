# Home & Navigation Module - Aryan

This module handles the main navigation, header, and home page components.

## Structure
- `components/` - Navigation and home UI components
- `hooks/` - Navigation-related hooks
- `services/` - Navigation services
- `utils/` - Navigation helper functions
- `styles/` - Module-specific styles

## Usage
```jsx
import { Header, Navbar, useNavigation } from '@/modules/home-navigation';
```

## Components
- Header
- Navbar
- Sidebar
- HomePage

## Hooks
- useNavigation
- useMenu
