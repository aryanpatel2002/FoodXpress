# Order Management Module - Prathamesh

This module handles all order-related functionality including creation, tracking, and management.

## Structure
- `components/` - Order management UI components
- `hooks/` - Order-related hooks
- `services/` - Order API services
- `utils/` - Order helper functions
- `styles/` - Module-specific styles

## Usage
```jsx
import { OrderList, useOrders } from '@/modules/order-management';
```

## Components
- OrderList
- OrderDetails
- CreateOrder
- OrderStatus

## Hooks
- useOrders
