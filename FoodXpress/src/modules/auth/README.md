# Auth Module - Janardhan

This module handles all authentication-related functionality.

## Structure
- `components/` - Authentication UI components
- `hooks/` - Custom hooks for auth logic
- `services/` - API calls and auth services
- `utils/` - Helper functions for authentication
- `styles/` - Module-specific styles

## Usage
```jsx
import { LoginForm, useAuth } from '@/modules/auth';
```

## Components
- LoginForm
- SignupForm
- ForgotPassword

## Hooks
- useAuth
- useLogin
