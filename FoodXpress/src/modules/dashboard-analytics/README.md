# Dashboard & Analytics Module - Mayur

This module handles dashboard views, analytics, charts, and reporting functionality.

## Structure
- `components/` - Dashboard and analytics UI components
- `hooks/` - Dashboard-related hooks
- `services/` - Analytics API services
- `utils/` - Dashboard helper functions
- `styles/` - Module-specific styles

## Usage
```jsx
import { Dashboard, Analytics, useDashboard } from '@/modules/dashboard-analytics';
```

## Components
- Dashboard
- Analytics
- Charts
- Reports

## Hooks
- useDashboard
- useAnalytics
